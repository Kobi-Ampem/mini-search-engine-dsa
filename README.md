# Mini Search Engine 🔍

## Description
This project is a mini search engine built using Python that demonstrates the use of data structures and algorithms for efficient text searching and retrieval.

## Features
- Inverted index
- Multi-keyword search
- Boolean queries (AND, OR, NOT)
- Ranking based on keyword frequency
- Snippet preview
- File upload support
- Streamlit UI

## Technologies Used
- Python
- Streamlit

## Project Structure
- engine/ → Core logic (indexing, searching)
- ui/ → Streamlit interface
- data/ → Sample text files
- utils/ → Helper functions

## How to Run
```bash
streamlit run ui/app.py
